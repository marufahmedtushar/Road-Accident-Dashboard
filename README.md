# Road-Accident-Dashboard
## Objective

Analyze accident trends and identify major causes.

## Dataset

66,000+ records

## Tools

Power BI
Excel

## KPIs

-Total Accidents

-Casualties

-Fatalities

-Road Type

-Weather

-Severity

## Insights

• Weekends have more accidents.
• Rain increases severe accidents.
• Urban roads have the highest casualty count.

• Total Accidents: 660,679
• Total Casualties: 896,568

• Accident frequency peaks in November.
• Saturday records the highest number of accidents.
• Most accidents occur during daylight rather than in darkness.
